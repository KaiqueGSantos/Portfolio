Sobre Mim

Olá, meu nome é Kaique tenho 18 anos, sou um programador com experiencia em Python, Desenvolvimento Web e Desenvolvimento de Jogos utilizando C# na Unity. Possuo fácil aprendizagem, objetividade, resiliência e estou sempre em busca de novos conhecimentos, os quais quero utilizar para ajudar as pessoas e empresas.

HTML5

HTML é uma abreviação de Hypertext Markup Language, ou seja, Linguagem de Marcação de Hipertexto. Resumindo, o HTML é uma linguagem usada para a publicação de conteúdo (texto, imagens, vídeos, áudio etc.) na web.

CSS3

CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site.

JavaScript

O JavaScript é uma linguagem de programação de alto nível voltada para o desenvolvimento web, criada originalmente para funcionar do lado do usuário, ou seja, nos navegadores. Junto do HTML e do CSS, é uma das principais tecnologias da web, permitindo a criação de páginas interativas com elementos dinâmicos e boa performance.

Python

O Python é uma linguagem de programação amplamente usada em aplicações da Web, desenvolvimento de software, ciência de dados e machine learning (ML). Os desenvolvedores usam o Python porque é eficiente e fácil de aprender e pode ser executada em muitas plataformas diferentes.

C#

C# é uma linguagem de programação orientada a objetos e orientada a componentes. C# fornece construções de linguagem para dar suporte diretamente a esses conceitos, tornando uma linguagem natural para criação e uso de componentes de software.

Unity

Unity é uma ferramenta que permite criar videojogos para diversas plataformas (PC, consoles, mobile, VR e AR) utilizando um editor visual e programação através de scripting (C#), oferecendo aos utilizadores ferramentas profissionais, capazes de preencher os requisitos de qualquer jogo.
